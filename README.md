# mymenuifymod
** Wii Theme Installer

- priiloader check
- built in brick protection
- Sd/Usb compatible


** Usage 

- Minus - button  - reload ios
- Plus + button - download and install original system menu theme .app file
- A button - select/(Un)install/delete
- B button - back/exit
- Home button = exit Hbc
- Left/Right - toggle selections
